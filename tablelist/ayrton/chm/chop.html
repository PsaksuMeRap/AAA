chop(ayrton)R Documentation

Remove last characters from strings

Description

This function removes the last "n" characters from a string or vector of strings.

Usage

chop(s,n)

Arguments

s a string or vector of strings
n the number of characters to remove

Value

The input strings with the last n characters removed.
